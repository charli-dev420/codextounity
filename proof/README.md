# Proof Directory

This directory is intentionally kept free of raw local proof files in the distributed plugin.

Validation scripts may write redacted proof reports here. Reports must use placeholders such as
`<PLUGIN_ROOT>`, `<CODEX_HOME>`, `<UNITY_PROJECT>`, `<WORK_DIR>` and `<INSTALL_ROOT>`.

Raw machine logs, screenshots and local validation artifacts must stay outside the packaged plugin.
