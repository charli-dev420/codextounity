# Troubleshooting Index / Index de depannage

Use the localized guides.

Utiliser les guides localises.

- French / Francais: `docs/fr/TROUBLESHOOTING.md`
- English / Anglais: `docs/en/TROUBLESHOOTING.md`

Before sharing logs or packaging, run:

Avant de partager des logs ou de preparer une publication, lancer :

```powershell
.\scripts\scan_private_leaks.ps1
```

The scan must pass and local logs must be redacted.

Le scan doit passer et les logs locaux doivent etre rediges.
